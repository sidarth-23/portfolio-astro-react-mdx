## Overview

This portfolio replaces a CMS-backed setup with a lighter, code-first approach:

- **Astro** for static site generation and partial hydration
- **MDX** for rich, component-embeddable content
- **Zod** for runtime validation of all frontmatter
- **Tailwind CSS + shadcn** for a minimal, consistent UI layer

## Goals

1. Keep the dependency surface small.
2. Make content authoring feel like writing code.
3. Ensure every data boundary is validated and typed.

## Stack

| Layer        | Tooling                              |
| ------------ | ------------------------------------ |
| Framework    | Astro                                |
| Content      | MDX (content collections)            |
| Styling      | Tailwind CSS v4 + shadcn/ui          |
| Validation   | Zod                                  |
| Icons        | Lucide React / HugeIcons            |
| Fonts        | Geist Variable                       |

## Result

A fast, typed, minimal portfolio that is easy to extend and pleasant to maintain.
