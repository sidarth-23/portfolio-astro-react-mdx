## Visión General

Cada backend necesita un frontend. Este dashboard de analíticas se conecta a tus APIs vía WebSocket y renderiza métricas en tiempo real, logs y datos de rendimiento en una interfaz limpia y responsiva.

## Stack Tecnológico

| Capa         | Tecnología                          |
| ------------ | ----------------------------------- |
| Framework    | React 19                            |
| Lenguaje     | TypeScript                          |
| Estilos      | Tailwind CSS + shadcn/ui            |
| Estado       | TanStack Query                      |
| Tiempo Real  | WebSocket + Server-Sent Events      |
| Gráficos     | Recharts                            |

## Características Principales

- **Métricas en Tiempo Real**: Gráficos en vivo de CPU, memoria y tasa de solicitudes
- **Streaming de Logs**: Cola de logs filtrable, con colores y búsqueda
- **Gestión de Alertas**: Configura umbrales y canales de notificación
- **Modo Oscuro**: Soporte completo de modo oscuro con detección de preferencia del sistema

## Arquitectura

El dashboard usa un sistema de widgets modular:

1. **Registro de Widgets**: Carga dinámica de widgets
2. **Fuentes de Datos**: Conectores enchufables (REST, WebSocket, GraphQL)
3. **Gestión de Estado**: Filtros sincronizados con URL y rangos de tiempo

## Resultados

Maneja 1000+ conexiones WebSocket concurrentes con renderizado fluido a 60fps.
