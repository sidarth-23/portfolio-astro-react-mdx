## Visión General

Construir microservicios en Go no debería requerir reinventar la rueda cada vez. Este toolkit proporciona las piezas fundamentales que necesitas: descubrimiento de servicios, circuit breakers, health checks y trazabilidad distribuida — todo conectado con valores predeterminados sensatos.

## Características

- **Descubrimiento de Servicios**: Registro automático con Consul
- **Circuit Breaker**: Previene fallos en cascada con umbrales configurables
- **Trazabilidad Distribuida**: Integración OpenTelemetry lista para usar
- **Health Checks**: Endpoints de salud HTTP y gRPC
- **Apagado Graceful**: Maneja SIGTERM correctamente con configuración de timeout

## Arquitectura

El toolkit sigue un enfoque por capas:

1. **Capa de Transporte** (HTTP/gRPC)
2. **Capa de Servicio** (Lógica de Negocio)
3. **Capa de Repositorio** (Acceso a Datos)
4. **Infraestructura** (Logging, Métricas, Trazabilidad)

## Primeros Pasos

```go
import "github.com/sidarth-23/go-microservices"

svc := micro.NewService(
    micro.Name("user-service"),
    micro.Version("1.0.0"),
)

svc.Init()
svc.Run()
```
