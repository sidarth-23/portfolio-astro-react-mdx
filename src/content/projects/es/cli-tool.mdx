## Visión General

¿Cansado de cambiar entre kubectl, docker y ssh? Esta CLI unifica operaciones comunes de DevOps en una sola herramienta rápida escrita en Go.

## Características

- **Despliegue**: Despliegues con un solo comando con soporte para rollback
- **Logs**: Streaming de logs desde múltiples servicios simultáneamente
- **Inspección**: Verifica salud del servicio, uso de recursos y configuraciones
- **Secretos**: Gestiona variables de entorno y secretos de forma segura

## Comandos

```bash
# Desplegar un servicio
devops deploy --service api --env production

# Stream de logs de todos los servicios
devops logs --all --follow

# Verificar salud del servicio
devops health --service api

# Ver uso de recursos
devops stats --service api
```

## Configuración

La CLI usa un archivo YAML para definiciones de entorno:

```yaml
environments:
  production:
    cluster: prod-k8s
    namespace: default
    registry: gcr.io/my-project
```

Reduce el tiempo de despliegue de 15 minutos a 2 minutos. Usada diariamente por un equipo de 8 ingenieros.
