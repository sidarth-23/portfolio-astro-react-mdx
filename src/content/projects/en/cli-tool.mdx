## Overview

Tired of switching between kubectl, docker, and ssh? This CLI unifies common DevOps operations into a single, fast tool written in Go.

## Features

- **Deploy**: One-command deployments with rollback support
- **Logs**: Stream logs from multiple services simultaneously
- **Inspect**: Check service health, resource usage, and configurations
- **Secrets**: Manage environment variables and secrets securely

## Commands

```bash
# Deploy a service
devops deploy --service api --env production

# Stream logs from all services
devops logs --all --follow

# Check service health
devops health --service api

# View resource usage
devops stats --service api
```

## Configuration

The CLI uses a YAML config file for environment definitions:

```yaml
environments:
  production:
    cluster: prod-k8s
    namespace: default
    registry: gcr.io/my-project
```

## Results

Reduces deployment time from 15 minutes to 2 minutes. Used daily by a team of 8 engineers.
