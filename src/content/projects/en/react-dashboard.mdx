## Overview

Every backend needs a frontend. This analytics dashboard connects to your APIs via WebSocket and renders real-time metrics, logs, and performance data in a clean, responsive interface.

## Tech Stack

| Layer        | Technology                          |
| ------------ | ----------------------------------- |
| Framework    | React 19                            |
| Language     | TypeScript                          |
| Styling      | Tailwind CSS + shadcn/ui            |
| State        | TanStack Query                      |
| Real-time    | WebSocket + Server-Sent Events      |
| Charts       | Recharts                            |

## Key Features

- **Real-time Metrics**: Live CPU, memory, and request rate charts
- **Log Streaming**: Filterable, color-coded log tail with search
- **Alert Management**: Configure thresholds and notification channels
- **Dark Mode**: Full dark mode support with system preference detection

## Architecture

The dashboard uses a modular widget system:

1. **Widget Registry**: Dynamic widget loading
2. **Data Sources**: Pluggable connectors (REST, WebSocket, GraphQL)
3. **State Management**: URL-synced filters and time ranges

## Results

Handles 1000+ concurrent WebSocket connections with smooth 60fps rendering.
